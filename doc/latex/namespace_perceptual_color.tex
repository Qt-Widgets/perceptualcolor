\doxysubsection{Detailed Description}
The namespace of this library. 

Everything that is provides in this library is encapsulated within this namespace.

The library uses generally {\ttfamily int} for integer values, because Q\+Size() also Q\+Point() also do, and the library relies heavily on the usage of Q\+Size() and Q\+Point(). It uses generally {\ttfamily qreal} for floating point values, because Q\+Point\+F() also does, for the same reasons as above.

The source code of the library is in U\+T\+F8. A static\+\_\+assert makes sure your compiler actually treats it also as U\+T\+F8.
