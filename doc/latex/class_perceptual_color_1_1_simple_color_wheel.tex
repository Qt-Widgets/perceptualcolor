Refresh the wheel ribbon pixmap. 

This class has a cache for the pixmap of the wheel ribbon.

This data is cached because it is often needed and it would be expensive to calculate it again and again on the fly.

Calling this function updates this cached data.

An update might be necessary if the data the pixmap relies on, has changed. For example, if the \mbox{\hyperlink{class_perceptual_color_1_1_simple_color_wheel_ace077e19741c0b12eda02ebd74847cea}{wheel\+Ribbon\+Chroma()}} property or the widget size have changed, this function has to be called.
