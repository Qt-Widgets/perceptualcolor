\doxysubsection{Detailed Description}
A slider displaying a gradient. 

A slider that displays a gradient between two L\+Ch colors. The gradient is calculated by equal steps in the L\+Ch color space. Concerning the h value (hue), which is circular, always the shorter side of the circle is choosen. Examples\+:  If the first hue is 182° and the second hue is 1°, than the hue will increase from 182° to 360° than 1°. If the first hue is 169° and the second hue is 359°, than the hue will decrease from 169° to 0°, than 359°.

This widget also renders the alpha channel, using a background of gray squares for colors that are not fully opaque.

Note that due to this mathematical model, there might be out-\/of-\/gamut colors within the slider even if both, the first and the second color are in-\/gamut colors. Out-\/of-\/gamut colors are not rendered, so you might see a hole in the gradient.

{\footnotesize\ttfamily Q\+Image Perceptual\+Color\+::\+Gradient\+Selector\+::m\+\_\+gradient\+Image\hspace{0.3cm}{\ttfamily [private]}}



Cache for the gradient image. 

Holds the current gradient image (without the selection cursor). Always at the left is the first calor, always at the right is the second color. So when painting, it might be necessary to rotate the image.

This is a cache. Before using it, check if it\textquotesingle{}s up-\/to-\/date with \mbox{\hyperlink{class_perceptual_color_1_1_gradient_selector_af68993ee22c8614f8b1237e60d016dc1}{m\+\_\+gradient\+Image\+Ready()}}. If not, use \mbox{\hyperlink{class_perceptual_color_1_1_gradient_selector_a82a87ec1701c4db18f821c3b35026e90}{update\+Gradient\+Image()}} to update it.

If something in the widget makes a new m\+\_\+gradien\+Image() necessary, do not directly call \mbox{\hyperlink{class_perceptual_color_1_1_gradient_selector_a82a87ec1701c4db18f821c3b35026e90}{update\+Gradient\+Image()}} but just set m\+\_\+gradient\+Image\+Ready to {\itshape false}. So it can be re-\/generated next time it\textquotesingle{}s actually used, and we do not waste C\+PU power for updating for example invisible widgets.
