\doxysubsection{Detailed Description}
Informations about L$\ast$a$\ast$b$\ast$ gamut boundaries. 

According to the \href{https://de.wikipedia.org/w/index.php?title=Lab-Farbraum&oldid=197156292}{\texttt{ German Wikipedia}}, in L$\ast$a$\ast$b$\ast$ color space, in typical software implementations are used usually these ranges\+:
\begin{DoxyItemize}
\item Lightness axis\+: 0..100
\item a axis\+: -\/128..127 (in C++ a signed 8 bit integer)
\item b axis\+: -\/128..127 (in C++ a signed 8 bit integer)
\end{DoxyItemize}

According to the same Wikipedia article, the physical range for a and b axis goes up to\+:
\begin{DoxyItemize}
\item Lightness axis\+: 0..100
\item a axis\+: -\/170..100
\item b axis\+: -\/100..150
\end{DoxyItemize}

in some situations.
