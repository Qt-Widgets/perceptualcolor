\doxysubsection{Detailed Description}
A point in the polar coordinate system. 

Defines a point in the two-\/dimensional polar coordinate system using floating point precision.

Polar coordinates allow multiple representations for a single point\+:
\begin{DoxyItemize}
\item An angle of~0° is the same as 360° is the same as 720° is the same as~−360°.
\item A radial of~1 and an angle of~0° is the same as a radial of~−1 and an angle of~180°.
\item If the radial is~0, the angle is meaningless.
\end{DoxyItemize}

This class automatically normalizes the values\+:
\begin{DoxyItemize}
\item the radial is normalized to value~≥~0
\item the angle is normalized to 0°~≤~value~$<$~360°
\end{DoxyItemize}

When the radial is~0, often by convention the (meaningless) angle is set also~0. This class does {\itshape not} normalize the angle to~0 when the radial is~0 as long as initialized with a radial and an angle. However, when initialized with cartesian coordinates (0,~0) then the result is radial~0 and angle~0. See also \mbox{\hyperlink{class_perceptual_color_1_1_polar_point_f_a4cad163898ab00cf9ebfd5a52d4d471f}{operator==()}}.
