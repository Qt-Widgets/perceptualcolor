Detailed Description

Informations about L*a*b* gamut boundaries.

According to the German Wikipedia, in L*a*b* color space, in typical software implementations are used usually these ranges:

  • Lightness axis: 0..100
  • a axis: -128..127 (in C++ a signed 8 bit integer)
  • b axis: -128..127 (in C++ a signed 8 bit integer)

According to the same Wikipedia article, the physical range for a and b axis goes up to:

  • Lightness axis: 0..100
  • a axis: -170..100
  • b axis: -100..150

in some situations.
