Get information from the ICC profile.

Fetches information from the ICC profile.

Parameters
profileHandlethe profile in which is searched
infoTypethe type of information that is searched
Returns
a QString with the information. It prefers the information in the current locale (country code and language code as used by QLocale()). If this is not available in the ICC profile, it silently falls back to an available localization. The returned QString() might be empty if the information is not available in the ICC profile.
