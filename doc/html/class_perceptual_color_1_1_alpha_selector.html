void PerceptualColor::AlphaSelector::registerAsBuddy ( QLabel *  label)

Register this widget as buddy.

This function registers this widget as the focus buddy of a QLabel. It's better using this function than calling directly QLabel::setBuddy(). This function does not set this entire widget as focus buddy, but only the spinbox part, which seems reasonable as focus buddies react on keyboard events and the spinbox is better adapted to keyboard input than the slider.
