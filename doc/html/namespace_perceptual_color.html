Detailed Description

The namespace of this library.

Everything that is provides in this library is encapsulated within this namespace.

The library uses generally int for integer values, because QSize() also QPoint() also do, and the library relies heavily on the usage of QSize() and QPoint(). It uses generally qreal for floating point values, because QPointF() also does, for the same reasons as above.

The source code of the library is in UTF8. A static_assert makes sure your compiler actually treats it also as UTF8.
